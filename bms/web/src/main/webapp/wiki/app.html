小程序校验文件

注册

通过 https://greenwishing.cn/mapping/register?key=aaaa.txt&content=bbbbbbb 注册后，可直接访问 https://greenwishing.cn/aaaa.txt 下载内容为 bbbbbbb 的文件

客户端OAuth2.0授权

创建应用

1.1 在应用模块创建应用，每个用户最多可创建3个

1.2 获取access_token

https://greenwishing.cn/oauth/token?client_id=应用ID&client_secret=应用密钥&grant_type=client_credentials

成功返回：

{
    "access_token":"14e46b68-282e-45e3-b98e-4babbb142066",
    "token_type":"bearer",
    "expires_in":7200,
    "scope":"read trust"
}

失败返回：

{
    "error":"unauthorized",
    "error_description":"No client with requested id: 应用ID"
}

1.3 获取最近几个月的汇总数据

https://greenwishing.cn/oauth/api/nearest?access_token=ACCESS_TOKEN&size=7

请求参数：1.2中获取的access_token，及需要返回汇总数据的大小size，size不传默认为20

返回数据格式：

{
